About Page

  • This Page was create by Benjamin Homa using the following:

    • AngularJS
    • Javascript
    • Bootstrap
    • Angular-Bootstrap
    • Google Maps API to display and plot routes on Maps

  • Code Structure is as follows

    • css directive holds main.css which binds to elements based on class
    • js directive holds the controller directive which is the controller for this project
    • node_modules was imported using Node.Js, Bootstrap contains bootstrap, Angular contains AngularJS and all package.json files
    • Open was used to open the localhost to port 8888 when user runes npm start
    • airports.Json is a JSON file containing data on airports around the globe, was only put in array based on airports inside the United States
